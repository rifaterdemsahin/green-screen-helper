# Green Screen Helper

This web application helps you identify imperfections in your green screen setup using your device's camera.

## How to Use

1.  Open the `index.html` file in a web browser.
2.  Click the "Start Camera" button to turn on your camera.
3.  Click on a spot on your green screen in the video feed to set the key color.
4.  Adjust the "Tolerance" slider to fine-tune the detection. Higher values are more lenient.
5.  The application will highlight areas that are not a good match for the key color, indicating potential issues like shadows, wrinkles, or uneven lighting.

## Features

*   **Real-time Analysis:** Get instant feedback on your green screen setup.
*   **Color Picker:** Accurately select your green screen's specific shade.
*   **Adjustable Tolerance:** Fine-tune the sensitivity of the imperfection detection.
*   **Clear Visualization:** Imperfections are highlighted, while the rest of the green screen is made transparent to help you focus on the issues.
